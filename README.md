# Personal Portfolio

Personal site. Not a template, not open for contributions.

**Live:** [denz-platform.vercel.app](https://denz-platform.vercel.app)

---

I built this to have somewhere that actually feels like me rather than a generic portfolio site. The idea was to make something visually interesting using nothing but vanilla HTML, CSS, and a canvas no frameworks, no build step, no dependencies. The orbital animation came from wanting the projects section to feel alive rather than just a list of cards. Each planet is a project.

The Interstellar theme is intentional. I like the film a lot, and the entry overlay with the TARS quote and ambient audio sets the tone for what the rest of the site tries to do: be a bit different.
